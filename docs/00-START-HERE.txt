💾 FILE MANAGEMENT
═══════════════════════════════════════════════════════════════════════════

BEFORE EDITING:
  • Make backup: copy ro-database.json to ro-database-backup.json
  • Use version control (git) if available

WHEN EDITING:
  • Use modern text editor (VS Code, Notepad++, Sublime)
  • Validate JSON before saving (jsonlint.com)
  • Keep ID numbering consistent
  • Follow naming conventions

AFTER EDITING:
  • Save the file
  • Refresh browser (Ctrl+F5)
  • Check for errors in console (F12)
  • Verify new data appears


🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

Problem: File won't open in browser
  → Right-click → Open With... → Select browser

Problem: Search returns no results
  → Check spelling (case-insensitive)
  → Try searching by ID instead
  → Ensure data is in ro-database.json

Problem: JSON looks like text
  → That's normal! Edit in text editor, not browser
  → It should be valid JSON format

Problem: Changes don't show in app
  → Save ro-database.json
  → Refresh browser (Ctrl+F5 or Cmd+Shift+R)
  → Check console for errors (F12)

Problem: Colors/fonts look wrong
  → Check browser compatibility
  → Try different browser
  → Clear browser cache
